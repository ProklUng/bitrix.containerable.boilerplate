# Boilerplate для работы с контейнерами Symfony в модулях Битрикс

***INTERNAL***

### Установка

composer.json:

```json
    "repositories": [
        {
            "type": "git",
            "url": "https://github.com/proklung/bitrix.containerable.boilerplate"
        }
    ]
```

```bash
composer require proklung/bitrix-containerable-boilerplate
```
